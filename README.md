# aixinge-ui

#### 介绍
Ai（智能）Xin（信）Ge（鸽）—— 智能消息推送系统 前端代码

#### 软件架构
基于Ant Design Pro Of Vue为模板架构实现


#### 安装依赖

```
yarn install
```

#### 开发模式运行
```
yarn run serve
```

####  编译项目
```
yarn run build
```

#### Lints and fixes files
```
yarn run lint
```
